Stock Market Analytics – Risk & Return Analysis

This project analyzes historical stock data to evaluate comparative risk and return profiles.

Metrics Used
- Daily returns
- Rolling volatility
- Moving averages
- Sharpe ratio
- Maximum drawdown
- Correlation analysis

Key Insights
- MSFT offers superior risk-adjusted performance
- AAPL provides strong downside protection
- NFLX exhibits high volatility with limited reward

Skills Demonstrated
Financial Markets · Risk Analysis · Time Series · Python · Data Visualization
